# Responsive Image Gallery 📸

A fully responsive **Image Gallery Project** built using **HTML, CSS, and JavaScript**.  
This project includes category filtering, hover effects, animations, and a lightbox preview with next/previous navigation.

---

## 🚀 Features

- Responsive grid layout
- Image hover zoom effect
- Category filtering
- Lightbox image preview
- Next & Previous image navigation
- Smooth animations
- Mobile-friendly design

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Functionalities

### ✅ Filter Images
Users can filter gallery images based on categories.

### ✅ Lightbox Preview
Clicking an image opens it in fullscreen mode.

### ✅ Navigation Controls
Users can:
- View next image
- View previous image
- Close the lightbox

### ✅ Responsive Design
The gallery automatically adjusts for:
- Desktop
- Tablet
- Mobile devices
